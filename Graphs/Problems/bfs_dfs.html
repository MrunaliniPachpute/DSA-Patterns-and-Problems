Breadth-First Search & Depth-First Search

Problem Description

BFS and DFS are two fundamental graph traversal algorithms:

  • BFS (Breadth-First Search): Explores the graph level by level, visiting all neighbors of a vertex before moving to the next level.
  • DFS (Depth-First Search): Explores the graph by going as deep as possible along each branch before backtracking.

Time Complexity:

  • BFS: O(V + E)
  • DFS: O(V + E)

Space Complexity:

  • BFS: O(V) for queue
  • DFS: O(V) for recursion stack
