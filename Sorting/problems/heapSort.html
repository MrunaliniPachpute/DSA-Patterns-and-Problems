Heap Sort

Algorithm Explanation

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It builds a max heap from the input data and repeatedly extracts the largest element from the heap to construct the sorted array. It is efficient and performs sorting in-place.

Pseudocode

function heapSort(array):
  buildMaxHeap(array)
  for i from length(array) - 1 down to 1:
    swap(array[0], array[i])
    heapify(array, 0, i)

function buildMaxHeap(array):
  for i from floor(length(array)/2) - 1 down to 0:
    heapify(array, i, length(array))

function heapify(array, i, n):
  largest = i
  left = 2*i + 1
  right = 2*i + 2
  
  if left < n and array[left] > array[largest]:
    largest = left
  if right < n and array[right] > array[largest]:
    largest = right

  if largest != i:
    swap(array[i], array[largest])
    heapify(array, largest, n)

Complexity & Properties

  • Time Complexity: O(n log n)
  • Space Complexity: O(1)
  • Stable: No
  • In-place: Yes
