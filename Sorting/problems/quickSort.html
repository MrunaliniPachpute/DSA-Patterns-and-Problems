Quick Sort

Algorithm Explanation

Quick Sort is an efficient, in-place, divide and conquer sorting algorithm. It works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays, according to whether they are less than or greater than the pivot. The sub-arrays are then sorted recursively. The choice of pivot and the partitioning scheme can vary, affecting the algorithm's performance.

Pseudocode (Lomuto Partition Scheme)

function quickSort(array, low, high):
  if low < high:
    pivotIndex = partition(array, low, high)
    quickSort(array, low, pivotIndex - 1)
    quickSort(array, pivotIndex + 1, high)

function partition(array, low, high):
  pivot = array[high]
  i = low - 1
  for j from low to high - 1:
    if array[j] <= pivot:
      i = i + 1
      swap(array[i], array[j])
  swap(array[i + 1], array[high])
  return i + 1

Complexity & Properties

  • Time Complexity: O(n log n) in the best and average case. O(nÂ²) in the worst case (e.g., when the array is already sorted and the last element is chosen as pivot).
  • Space Complexity: O(log n) on average (for recursion stack), O(n) in the worst case.
  • Stable: No, it is not a stable sort.
