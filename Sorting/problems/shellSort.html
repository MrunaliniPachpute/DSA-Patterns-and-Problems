Shell Sort

Algorithm Explanation

Shell Sort is an optimization over Insertion Sort that allows the exchange of far-apart elements. It works by sorting elements at specific intervals (gaps), which are gradually reduced until the array becomes fully sorted. It improves performance significantly for moderately sized datasets.

Pseudocode

function shellSort(array):
  n = length(array)
  gap = n / 2
  while gap > 0:
    for i = gap to n - 1:
      temp = array[i]
      j = i
      while j >= gap and array[j - gap] > temp:
        array[j] = array[j - gap]
        j = j - gap
      array[j] = temp
    gap = gap / 2

Complexity & Properties

  • Time Complexity: O(n log n) to O(n¬≤), depending on gap sequence.
  • Space Complexity: O(1)
  • Stable: No
  • In-place: Yes
