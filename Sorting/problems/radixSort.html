Radix Sort

Algorithm Explanation

Radix Sort is a non-comparative sorting algorithm that sorts numbers by processing individual digits. It sorts elements by each digit from least significant to most significant using a stable sorting method like Counting Sort. It works efficiently for fixed-length integer or string keys.

Pseudocode

function radixSort(array):
  maxNum = getMax(array)
  exp = 1
  while maxNum / exp > 0:
    countingSort(array, exp)
    exp = exp * 10

function countingSort(array, exp):
  n = length(array)
  output = array of size n
  count = array[0...9] initialized to 0

  for i = 0 to n - 1:
    index = (array[i] / exp) % 10
    count[index]++

  for i = 1 to 9:
    count[i] += count[i - 1]

  for i = n - 1 down to 0:
    index = (array[i] / exp) % 10
    output[count[index] - 1] = array[i]
    count[index]--

  copy output[] to array[]

Complexity & Properties

  • Time Complexity: O(n¬∑k), where k is the number of digits.
  • Space Complexity: O(n + k)
  • Stable: Yes
  • In-place: No
