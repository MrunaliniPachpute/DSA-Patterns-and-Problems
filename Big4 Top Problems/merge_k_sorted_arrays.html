Merge K Sorted Arrays

Hard

You are given $k$ sorted arrays. Merge them into a single sorted array.

This problem is a foundational question that tests your knowledge of **Heaps (Priority Queues)**, which are crucial for maintaining order efficiently across multiple sources.

Example:

Input: arrays = [[1, 4, 5], [1, 3, 4], [2, 6]] Output: [1, 1, 2, 3, 4, 4, 5, 6]

Optimal Approach: Min Heap (Priority Queue)

The brute-force solution (merging arrays sequentially or copying all elements and sorting) is inefficient. The **Min Heap** approach is optimal.

A Min Heap is used to keep track of the **smallest element currently available** from all $k$ arrays. The heap will only ever store $k$ elements (one from each array at most), ensuring fast operations.

Time Complexity: $O(N \log k)$, where $N$ is the total number of elements across all arrays and $k$ is the number of arrays. Each of the $N$ elements is pushed and popped exactly once from a heap of size $k$, taking $O(\log k)$ time per operation.

Steps:

  1. Create a **Min Heap** and initialize it by pushing the first element from all $k$ arrays. The element pushed should store three pieces of information: **value**, **array index**, and **next element index** within that array.
  2. While the heap is **not empty**:
    • Extract the minimum element (the root of the Min Heap). Add its value to the result list.
    • Using the stored array index and element index, determine the next element in the same array.
    • If the array still has elements remaining, insert the next element into the Min Heap.

Conceptual Solution (Example Structure):

# Conceptual data structure for the heap element: (value, array_index, element_index)

def mergeKSortedArrays(arrays):
    min_heap = MinHeap()
    result = []
    
    # 1. Initialize heap with the first element of each array
    for i in range(len(arrays)):
        if arrays[i]:
            # Push (value, array_index, next_element_index)
            min_heap.push( (arrays[i][0], i, 1) ) 

    # 2. Extract min and insert next element
    while not min_heap.is_empty():
        val, arr_idx, ele_idx = min_heap.pop()
        result.append(val)

        # Check if there is a next element in the array
        if ele_idx < len(arrays[arr_idx]):
            next_val = arrays[arr_idx][ele_idx]
            min_heap.push( (next_val, arr_idx, ele_idx + 1) )
            
    return result
