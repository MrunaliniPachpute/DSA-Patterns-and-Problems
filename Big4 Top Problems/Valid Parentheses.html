Valid Parentheses

Easy/Medium

Given a string `s` containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

  1. Open brackets must be closed by the same type of brackets.
  2. Open brackets must be closed in the correct order.
  3. Every close bracket has a corresponding open bracket of the same type.

Example 1:

Input: s = "()" Output: true

Example 2:

Input: s = "()[]{}" Output: true

Example 3:

Input: s = "(]" Output: false

Constraints:

  • $1 \le \text{s.length} \le 10^4$
  • `s` consists of parentheses only '()[]{}'.

Approach: Stack

This problem is perfectly suited for a **Stack** data structure. The stack helps us keep track of pending *open* brackets and ensures that when a *closing* bracket is encountered, it matches the most recently opened, unclosed bracket.

Time Complexity: $O(N)$ (linear time) as we iterate through the string once. **Space Complexity:** $O(N)$ in the worst case (e.g., "((("), where $N$ is the length of the string, to store the stack.

Steps:

  1. Initialize an empty stack.
  2. Iterate through the string character by character:
    • If the character is an **opening bracket** ('(', '{', '['), push it onto the stack.
    • If the character is a **closing bracket** (')', '}', ']'):
      • Check if the stack is empty. If it is, the closing bracket has no corresponding open bracket, so return `False`.
      • Pop the top element from the stack. Check if it is the corresponding opening bracket for the current closing bracket. If it doesn't match, return `False`.
  3. After iterating through the whole string, if the stack is empty, it means all opening brackets were correctly closed. Return `True`. Otherwise, return `False`.

Example Solution (Conceptual Python):

def isValid(s):
    stack = []
    # Map for quick lookup of corresponding open/close pairs
    mapping = {")": "(", "}": "{", "]": "["}

    for char in s:
        # If it is a closing bracket
        if char in mapping:
            # Get the top element if stack is not empty, else use a placeholder '#'
            top_element = stack.pop() if stack else '#'

            # Check for a mismatch (or if stack was empty)
            if mapping[char] != top_element:
                return False
        else:
            # If it is an opening bracket, push it onto the stack
            stack.append(char)

    # The string is valid if the stack is empty at the end
    return not stack
