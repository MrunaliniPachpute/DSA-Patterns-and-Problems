11. 3Sum

Medium

Given an integer array `nums`, return all the **unique triplets** `[a, b, c]` such that `a + b + c = 0`.

Notice that the solution set must not contain duplicate triplets.

Example:

Input: nums = [-1, 0, 1, 2, -1, -4] Output: [[-1, -1, 2], [-1, 0, 1]] Explanation: (-1) + 0 + 1 = 0 (-1) + (-1) + 2 = 0

Constraints:

  • $3 \le \text{nums.length} \le 5000$
  • $-10^5 \le \text{nums}[i] \le 10^5$

Approach: Sorting + Two Pointers

The most efficient approach is to first **sort the array**. Then, iterate through the array, fixing the first element (`a`), and use the **Two Pointers** technique to find the remaining two elements (`b` and `c`) that sum up to `-a` (or `target - a`). Skip duplicate elements for `a`, `b`, and `c` to ensure unique triplets.

This approach transforms the $O(n^3)$ brute-force solution into an efficient $O(n^2)$ solution.

Steps:

  1. Sort the input array `nums`.
  2. Iterate from the first element up to `n-2` (since we need three elements). This is the fixed pointer `i`.
  3. If the current element `nums[i]` is the same as the previous one (`nums[i-1]`), skip it (to avoid duplicate triplets).
  4. Initialize the two pointers: `left = i + 1` and `right = n - 1`.
  5. While `left < right`:
    • Calculate `current_sum = nums[i] + nums[left] + nums[right]`.
    • If `current_sum == 0`, a valid triplet is found. Store it. Then, increment `left` and decrement `right`, while skipping any duplicates for both pointers.
    • If `current_sum < 0`, increment `left` to increase the sum.
    • If `current_sum > 0`, decrement `right` to decrease the sum.

Example Solution (Conceptual Python):

def threeSum(nums):
    nums.sort()
    result = []
    n = len(nums)

    for i in range(n - 2):
        # Skip duplicates for the first element
        if i > 0 and nums[i] == nums[i-1]:
            continue

        left, right = i + 1, n - 1
        target = -nums[i]

        while left < right:
            current_sum = nums[left] + nums[right]

            if current_sum == target:
                result.append([nums[i], nums[left], nums[right]])

                # Skip duplicates for the second and third elements
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                
                left += 1
                right -= 1

            elif current_sum < target:
                left += 1
            else:
                right -= 1
                
    return result
