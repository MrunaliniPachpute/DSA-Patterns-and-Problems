Single Non-Duplicate Element

LeetCode #540 [Medium]

Problem Statement

You are given a sorted array nums where every element appears twice except for one element which appears only once. Find and return the single element that appears only once.

Example: nums = [1,1,2,3,3,4,4,8,8] â†’ Output: 2

Approach: Binary Search on Pairs

Explanation: In a sorted array where every number appears twice except one, we can use binary search to find the single element efficiently. Before the single element, pairs start at even indices (0, 2, 4...), and after the single element, pairs start at odd indices (1, 3, 5...).

Time Complexity: O(log n)

Space Complexity: O(1)

function findSingleElement(nums):
    left = 0           
    right = length(nums) - 1    

    while left < right:       
        mid = (left + right) / 2 

        if mid is odd: 
            mid = mid - 1

        if nums[mid] == nums[mid + 1]:
            left = mid + 2
            
        else:
            right = mid

    return nums[left]
