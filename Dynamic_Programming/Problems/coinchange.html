Coin Change

LeetCode #322 [Medium]

Problem Statement

You are given an integer array coins representing different denominations and an integer amount representing a total amount of money.

Return the fewest number of coins needed to make up that amount. If it is not possible to make up that amount, return -1. 

Input: coins = [1,2,5], amount = 11
Output: 3

Approach 1: Recursion

function coinChange(coins, amount):
    if amount == 0: return 0
    if amount < 0: return INF
    res = INF
    for c in coins:
        res = min(res, 1 + coinChange(coins, amount - c))
    return res

Approach 2: Memoization

dp = array of size amount+1 filled with -1
function coinChange(coins, amount):
    if amount == 0: return 0
    if amount < 0: return INF
    if dp[amount] != -1: return dp[amount]
    res = INF
    for c in coins:
        res = min(res, 1 + coinChange(coins, amount - c))
    dp[amount] = res
    return dp[amount]

Approach 3: Tabulation


int coinChange(vector<int>& coins, int amt) {
    int n = coins.size();
    vector<vector<int>> dp(n + 1, vector<int>(amt + 1, INT_MAX - 1));
    for (int i = 0; i <= n; i++) dp[i][0] = 0;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= amt; j++) {
            dp[i][j] = dp[i - 1][j];
            if (coins[i - 1] <= j) {
                dp[i][j] = min(dp[i][j], 1 + dp[i][j - coins[i - 1]]);
            }
        }
    }
    return dp[n][amt] >= INT_MAX - 1 ? -1 : dp[n][amt];
}
