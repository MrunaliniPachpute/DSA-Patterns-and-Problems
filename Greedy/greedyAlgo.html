WONDERING, HOW DOES IT COVER PATTERNS?
This set of problems covers the most greedy algorithm patterns asked in coding interviews:

Activity Selection → Interval scheduling, choosing maximum non-overlapping tasks.
Fractional Knapsack → Value/weight ratio sorting, optimal resource allocation.
Job Sequencing with Deadlines → Task prioritization under constraints.
Huffman Coding → Optimal encoding using greedy tree building.
Minimum Coins → Greedy choice property in currency problems.

Together, these ensure you understand interval scheduling, ratio-based selection, priority-based allocation, greedy tree construction, and coin problems which are fundamental greedy patterns.
