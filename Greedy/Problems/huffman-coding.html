Huffman Coding

Problem Statement

Given frequencies of characters, construct a prefix code (Huffman code) to minimize the average code length of characters.

Brute Force Approach

Generate all possible prefix codes and select the one with minimum weighted path length. Extremely inefficient.

generate all prefix-free codes
for each code assignment:
    calculate weighted path length
return minimum

Better Approach

Sort characters by frequency and repeatedly combine the two smallest. But if sorting is done again and again, it becomes inefficient.

while more than one character:
    sort frequencies
    pick two smallest
    merge them into a new node
    push back into list

Optimal Approach (Heap)

Use a min-heap to efficiently extract smallest frequencies and build the tree.

make minHeap of frequencies
while heap has more than one:
    left = extractMin
    right = extractMin
    newNode = left+right
    insert newNode
generate codes by traversing tree
