Fractional Knapsack

Problem Statement

Given weights and values of items, put items in a knapsack of capacity W to get the maximum total value. You can take fractions of items.

Brute Force Approach

Generate all subsets of items and try to take full/fractional items to maximize value. This is exponential and infeasible for large n.

function bruteForceKnapsack(items, W):
    try all subsets of items
    for each item:
        take full, fraction, or skip
    return max value

Better Approach

Sort items by value and try placing them greedily without considering value/weight ratio. This does not always give the best answer.

sort items by value descending
for each item:
    if weight fits:
        take it
    else:
        take fraction

Optimal Approach (Greedy)

Sort items by value/weight ratio and take as much as possible from highest ratio first.

sort items by value/weight ratio
for each item in sorted order:
    if item fits fully:
        take full
    else:
        take fraction
        break
return max value
