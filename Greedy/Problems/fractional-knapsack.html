Fractional Knapsack

Problem Statement

Given weights and values of n items, put items in a knapsack of capacity W to maximize total value. You can take fractions of items.

Example: Values = [60, 100, 120], Weights = [10, 20, 30], W = 50
Output: 240 (Take 10+20+20 fraction of 30)

Brute Force Approach

Try all combinations of items to find the maximum value.

Time Complexity: O(2^n), Space Complexity: O(1)

generate all subsets of items
calculate total weight and value for each
return max value with weight <= W

Better Approach

Sort items by weight and attempt different packings.

Time Complexity: O(nÂ²), Space Complexity: O(n)

Optimal Approach (Greedy)

Sort items by value/weight ratio, pick items in order until knapsack is full. Take fraction of the last item if needed.

Time Complexity: O(n log n), Space Complexity: O(1)

calculate value/weight ratio for each item
sort items by ratio descending
take items until capacity full
take fraction if last item exceeds remaining capacity
return total value
