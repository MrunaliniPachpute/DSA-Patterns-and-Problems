Fractional Knapsack

Problem Statement

Given weights and values of items and a knapsack capacity W, find the maximum value that can be obtained by putting items in the knapsack. Items can be broken into fractions.

Example:
Input: W = 50, items = [(value=60, weight=10), (value=100, weight=20), (value=120, weight=30)]
Output: 240.0

Approach: Greedy by Value/Weight Ratio

Explanation: Sort items by value/weight ratio in descending order. Take as much of the highest ratio item as possible, then move to next.

Time Complexity: O(n log n)

Space Complexity: O(1)

sort items by value/weight ratio descending
for each item:
    if weight fits in knapsack:
        take full item
    else:
        take fraction that fits
return total value
