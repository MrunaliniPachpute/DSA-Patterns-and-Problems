Job Scheduling with Deadlines

Problem Statement

You are given n jobs, each job has a deadline and a profit. Each job takes 1 unit of time to complete. The task is to find the maximum profit you can earn by scheduling jobs such that each job is completed on or before its deadline.

Brute Force Approach

Generate all possible subsets of jobs, check if the deadlines can be satisfied, and compute profit. This ensures correctness but runs in O(2n) time. 

for each subset of jobs:
    if deadlines satisfied:
        profit = sum of job profits
        update max profit
return max profit

Better Approach

Sort jobs by profit and try to place each job in the latest available slot before its deadline. This takes O(n * maxDeadline) time. 

sort jobs by profit (descending)
slots = array of size maxDeadline initialized to empty
for each job in sorted list:
    for t = job.deadline downto 1:
        if slots[t] is empty:
            assign job to slots[t]
            break
return total profit of scheduled jobs

Optimal Approach

Use a Disjoint Set Union (DSU) or Union-Find to optimize slot finding. This reduces scheduling from O(n * maxDeadline) to O(n log n). 

sort jobs by profit (descending)
initialize parent array for DSU with size maxDeadline
function find(x):
    if parent[x] == x: return x
    parent[x] = find(parent[x])
    return parent[x]

for each job in sorted list:
    available = find(job.deadline)
    if available > 0:
        assign job to available
        union(available, available-1)
return total profit of scheduled jobs
