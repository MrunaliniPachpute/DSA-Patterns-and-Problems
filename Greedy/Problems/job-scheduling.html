Job Scheduling with Deadlines and Profits

Problem Statement

Given n jobs with deadlines and profits. Each job takes 1 unit of time. Schedule jobs to maximize total profit if only one job can be scheduled at a time.

Example:
Jobs = [(id=1, deadline=2, profit=100), (id=2, deadline=1, profit=19), (id=3, deadline=2, profit=27), (id=4, deadline=1, profit=25), (id=5, deadline=3, profit=15)]
Output: Jobs done = 2, Max Profit = 127

Approach: Greedy by Profit

Explanation: Sort jobs by profit in descending order. For each job, place it in the latest available slot before its deadline.

Time Complexity: O(n log n + n*d)

Space Complexity: O(d)

sort jobs by profit descending
for each job in sorted list:
    find empty slot before job.deadline
    if found:
        assign job
return total profit
