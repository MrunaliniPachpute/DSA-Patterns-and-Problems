Minimum Number of Platforms

Problem Statement

Given arrival and departure times of trains, find the minimum number of railway platforms needed so that no train has to wait.

Example:
Arrival = [900, 940, 950, 1100, 1500, 1800]
Departure = [910, 1200, 1120, 1130, 1900, 2000]
Output: 3

Approach: Sort and Two Pointers

Explanation: Sort arrivals and departures. Use two pointers to simulate train arrival/departure and track maximum trains at a time.

Time Complexity: O(n log n)

Space Complexity: O(1)

sort arrival and departure
platforms = 0, max_platforms = 0
i = 0, j = 0
while i < n and j < n:
    if arrival[i] <= departure[j]:
        platforms++
        i++
        max_platforms = max(max_platforms, platforms)
    else:
        platforms--
        j++
return max_platforms
