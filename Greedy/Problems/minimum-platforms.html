Minimum Number of Platforms

Problem Statement

Given arrival and departure times of trains, find the minimum number of platforms required so that no train waits.

Brute Force Approach

For each train, check how many other trains overlap with it and update the maximum. This results in O(nÂ²) time.

sort arrivals
sort departures

for i = 0 to n-1:
    count = 1
    for j = i+1 to n-1:
        if arrivals[j] < departures[i]:
            count++
    update maxPlatforms = max(maxPlatforms, count)

return maxPlatforms

Better Approach

Sort arrival times and for each train check the number of trains currently on the station using a line sweep.

sort arrivals and departures
for each arrival:
    if arrival < earliest departure:
        need new platform
    else:
        reuse platform

Optimal Approach

Use two pointers: one for arrivals, one for departures. Move whichever is smaller and track count of trains at station.

while i < n and j < n:
    if arrivals[i] <= departures[j]:
        count++
        max = max(max, count)
        i++
    else:
        count--
        j++
return max
