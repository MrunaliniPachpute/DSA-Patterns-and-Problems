Minimum Number of Platforms

Problem Statement

Given arrival and departure times of trains at a station, find the minimum number of platforms required so that no train waits.

Example: Arrival = [9:00, 9:45, 9:55], Departure = [9:30, 10:00, 11:00] → Output: 2 platforms

Brute Force Approach

For each train, check how many trains are at the station at the same time and take the maximum.

Time Complexity: O(n²), Space Complexity: O(1)

for i = 0 to n-1:
    count = 1
    for j = 0 to n-1, j != i:
        if arrival[j] <= departure[i] and departure[j] >= arrival[i]:
            count += 1
    update max_platforms
return max_platforms

Better Approach

Sort arrival and departure times and use two pointers to check overlaps.

Time Complexity: O(n log n), Space Complexity: O(n)

Optimal Approach (Greedy)

Sort both arrival and departure times separately. Traverse both arrays: - If next train arrives before the previous departs, increment platform count. - Else, decrement count as train departs.

Time Complexity: O(n log n), Space Complexity: O(1)

sort arrival[]
sort departure[]
platform_needed = 1
max_platforms = 1
i = 1, j = 0
while i < n and j < n:
    if arrival[i] <= departure[j]:
        platform_needed += 1
        i += 1
    else:
        platform_needed -= 1
        j += 1
    max_platforms = max(max_platforms, platform_needed)
return max_platforms
