Autocomplete / Suggestions

Problem Statement

Given a prefix, return all words from the trie that start with this prefix (autocomplete feature).

Example: Prefix = "app", Words = ["apple","app","application"] â†’ ["apple","app","application"]

Approach: Trie DFS

Explanation: Traverse trie to prefix node, then DFS from that node to collect all words.

Time Complexity: O(P + K*L), P = prefix length, K = #words with prefix, L = avg word length

Space Complexity: O(N*L) for trie

flowchart TD A[Start] --> B[Initialize result list] B --> C[Traverse trie to prefix node] C --> D[Initialize node = root] D --> E[For each character in prefix] E --> F{Character exists in node.children?} F -->|No| G[Return empty list] F -->|Yes| H[Move to child node] H --> I{More characters in prefix?} I -->|Yes| E I -->|No| J[Start DFS from prefix node] J --> K[Call DFS with prefix node and current word] K --> L{Node is end of word?} L -->|Yes| M[Add word to result list] L -->|No| N[Continue DFS] M --> N N --> O[For each child of current node] O --> P[Add child character to word] P --> Q[Recursively call DFS on child] Q --> R[Remove last character from word] R --> O O -->|No more children| S[Return from DFS] S --> T[Return result list] G --> T T --> U[End] style A fill:#e1f5fe style U fill:#e8f5e8 style F fill:#fff3e0 style I fill:#fff3e0 style L fill:#fff3e0
